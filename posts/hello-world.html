Hello, World!

Welcome to Blogo — a local-first, GitHub-powered dev blog. This is the first post, published as a plain HTML file directly from the in-browser PWA editor.

How It Works

Posts are written in an Editor.js block editor, auto-saved to IndexedDB as drafts, and published by pushing HTML files to a GitHub repository via the Contents API.

Tech Stack

This blog is built with:

  • Next.js 14 (App Router, SSG)
  • Tailwind CSS + shadcn/ui for styling
  • Editor.js for the in-browser CMS
  • Dexie.js (IndexedDB) for offline draft storage
  • GitHub Contents API for publishing

A Code Sample

// lib/posts.ts
export async function getAllPosts(): Promise<PostMeta[]> {
  const files = fs.readdirSync(postsDir).filter(f => f.endsWith('.json'))
  return files
    .map(f => JSON.parse(fs.readFileSync(path.join(postsDir, f), 'utf-8')))
    .sort((a, b) => new Date(b.publishedAt).getTime() - new Date(a.publishedAt).getTime())
}
